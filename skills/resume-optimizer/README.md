# 简历优化

简历优化工具能分析简历内容，从结构、措辞、亮点呈现等维度进行优化。支持匹配目标岗位要求，突出相关经验和技能，提升简历竞争力。

## 使用场景

- 求职前优化简历，匹配目标岗位的关键词和要求
- 改善简历措辞，将平淡描述转化为量化的成果导向表达
- 调整简历结构和排版建议，提升可读性
- 针对不同岗位定制简历版本

## 特点

- **岗位匹配**: 分析目标 JD，突出匹配的经验和技能
- **措辞优化**: 将职责描述转化为 STAR 法则的成果导向表达
- **关键词植入**: 自动识别行业关键词，提升 ATS 系统通过率
- **结构建议**: 优化简历板块顺序和内容比重

## 输出格式

```markdown
## 优化建议总览
- 整体评分及核心改进点

## 逐条优化
### 原文
> 原始描述

### 优化后
> 优化后的描述

### 优化说明
- 修改理由和技巧说明

## 补充建议
- 建议新增的内容或板块
- 建议删除或弱化的内容
```
