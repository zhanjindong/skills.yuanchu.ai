# 简历优化 — Claude Code

## 使用方法

在 Claude Code 中输入 `/resume-optimizer` 即可触发。提供简历内容和目标岗位信息，自动生成优化建议。

---

分析用户的简历内容，从结构、措辞、关键词等维度进行优化，匹配目标岗位要求。

## 步骤

1. 接收用户提供的简历内容和目标信息：
   - **简历内容**: 完整简历文本或特定板块
   - **目标岗位**: 岗位名称或 JD 描述（可选）
   - **优化重点**: 全面优化 / 措辞优化 / 结构优化 / 关键词优化
2. 分析当前简历：
   - 评估整体结构和内容比重
   - 识别措辞问题（职责导向 vs 成果导向）
   - 检查关键词覆盖度
3. 如果提供了目标 JD，进行匹配度分析：
   - 提取 JD 中的核心要求和关键词
   - 标注简历中匹配和缺失的部分
4. 逐条输出优化建议，包含原文对比和修改说明。

## 规则

- 优化后的描述遵循 STAR 法则（情境-任务-行动-结果）
- 尽可能量化成果（数字、百分比、规模）
- 保持真实性，不编造经历或夸大成果
- 关键词植入要自然，不堆砌
- 针对不同行业调整措辞风格
- 敏感个人信息（姓名、电话等）不在输出中重复
