# 技术文章生成

技术文章生成器是一个自动化的 AI 技术写作工具。它能自动搜索 AI、大模型、Agent 等前沿领域的最新动态，发现最值得写的热点话题，然后生成一篇深入浅出的技术文章 HTML 页面。支持两种模式：全自动模式（自动发现话题）和指定主题模式。

文章写作风格参考刘润——逻辑锋利、语言口语化、观点一针见血。核心理念是"把复杂的技术，写成你坐在朋友对面听他聊的感觉"，每个观点都要"写透"（讲故事 + 举例子 + 打比方 + 给金句）。

## 使用场景

- 定期产出 AI 领域技术博客文章，保持内容更新
- 围绕特定技术主题（如 MCP 协议、RAG、Agent 框架）撰写深度分析
- 自动发现本周最值得写的 AI 热点，省去选题时间
- 为技术社区、公众号、个人博客生成高质量原创内容

## 特点

- **双模式选题**: 全自动热点发现 + 指定主题深度搜索，灵活适配不同需求
- **风格多样性引擎**: 每篇文章随机组合开头模式、叙事弧线、结尾模式，避免千篇一律
- **深入浅出写作法**: 每个观点用故事、例子、比方、金句包裹，让非专业读者也能理解
- **SVG 矢量图表**: 内联 SVG 绘制架构图、流程图、对比图，视觉清晰且响应式
- **完整 HTML 输出**: 单文件 HTML 页面，含目录导航、代码高亮、多种组件（callout、表格、引用块等）
- **自动去重**: 检查已有文章列表，避免重复选题
- **自检机制**: 生成后自动验证目录锚点、CSS 完整性、写作质量等 13 项指标

## 输出格式

生成单文件 HTML 技术文章页面，包含：
- 文章元信息（标题、作者、日期、标签）
- 目录导航（锚点跳转）
- 正文章节（含代码块、callout 提示、SVG 图表、表格等组件）
- 参考资料列表（带超链接）

文件命名：`[slug-title].html`（如 `mcp-protocol-explained.html`）
