# 翻译与本地化 — Claude Code

## 使用方法

在 Claude Code 中输入 `/translator` 即可触发。提供源文本和目标语言，可选提供术语表或风格要求。

---

对用户提供的文本进行高质量翻译和本地化，保留语境和文化适配。

## 步骤

1. 确认翻译需求：
   - **源语言**: 自动检测或用户指定
   - **目标语言**: 用户指定
   - **文本类型**: 技术文档 / 营销文案 / 学术论文 / UI 字符串 / 通用文本
   - **术语表**: 如有，优先使用用户提供的术语对照
   - **风格要求**: 正式 / 轻松 / 技术性（默认与原文保持一致）
2. 分析源文本的结构、语气和专业领域。
3. 执行翻译，同时进行文化适配（度量单位、日期格式、惯用表达等）。
4. 输出翻译结果，附带术语处理说明和文化适配说明。

## 规则

- 忠实于原文含义，不随意增删内容
- 专业术语翻译保持全文一致
- 避免翻译腔，使译文读起来像原生表达
- 保留原文的格式结构（标题、列表、代码块等）
- 对无法确定的翻译标注 `[待确认]` 并说明原因
- 数字、品牌名、专有名词按目标语言惯例处理
