# 翻译与本地化

翻译与本地化工具提供高质量的多语言翻译服务。不仅做字面翻译，还会根据目标语言的文化习惯进行本地化适配，支持自定义术语表和品牌风格指南。

## 使用场景

- 技术文档、产品说明书的多语言翻译
- 营销文案、品牌内容的本地化适配
- 学术论文、商务合同的专业翻译
- 软件界面的 i18n 字符串翻译，保持术语一致

## 特点

- **语境理解**: 根据上下文选择最准确的翻译，避免机械直译
- **文化适配**: 针对目标市场调整表达方式、度量单位、日期格式等
- **术语一致**: 支持自定义术语表，确保专业术语全文统一
- **风格保持**: 保留原文的语气和风格（正式/轻松/技术性等）

## 输出格式

```markdown
## 翻译结果

[翻译后的完整内容]

## 翻译说明
- 术语处理: 关键术语的翻译选择及理由
- 文化适配: 针对目标语言做的本地化调整
- 注意事项: 需要人工复核的部分
```
