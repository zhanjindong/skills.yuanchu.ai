# 小红书文案 — Claude Code

## 使用方法

在 Claude Code 中输入 `/xiaohongshu-writer` 即可触发。描述产品/主题和目标受众，自动生成小红书风格文案。

---

根据用户提供的产品或主题信息，生成符合小红书平台风格的种草文案。

## 步骤

1. 确认文案需求：
   - **产品/主题**: 要推广或分享的内容
   - **目标受众**: 针对的人群画像
   - **笔记类型**: 种草推荐 / 经验分享 / 教程攻略 / 测评对比
   - **卖点/亮点**: 核心优势或关键信息
2. 生成 3-5 个备选标题，运用以下技巧：
   - 数字吸引（"5 个方法"、"30 天亲测"）
   - 悬念引导（"后悔没早知道"、"居然还有人不知道"）
   - 痛点共鸣（"告别 xxx 的烦恼"）
3. 生成正文内容，遵循爆款结构：
   - 开头：痛点引入 / 场景代入
   - 中间：产品介绍 / 使用体验 / 干货分享
   - 结尾：总结推荐 / 互动引导
4. 穿插 emoji 排版，生成话题标签。

## 规则

- 语气亲切自然，像朋友分享，避免硬广感
- 标题控制在 20 字以内，正文 300-800 字
- emoji 使用自然，不堆砌，每段 1-3 个
- 标签 5-10 个，包含热门标签和精准长尾标签
- 不使用虚假宣传或违规词汇
- 内容真实有价值，不做纯营销导向
