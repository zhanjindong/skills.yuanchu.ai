# 小红书文案

小红书文案生成器能根据产品或主题，生成符合小红书平台风格的种草文案。包含吸睛标题、正文内容、话题标签和 emoji 排版，帮助快速产出高质量的小红书笔记。

## 使用场景

- 品牌推广需要产出小红书种草笔记
- 个人博主分享好物、经验、教程类内容
- 电商运营需要批量生成产品推荐文案
- 内容营销团队的小红书渠道内容生产

## 特点

- **标题党**: 自动生成 3-5 个吸睛标题供选择，含数字、悬念、对比等技巧
- **爆款结构**: 遵循小红书爆款笔记结构（痛点引入 → 解决方案 → 使用体验 → 总结推荐）
- **emoji 排版**: 自动穿插合适的 emoji，符合小红书阅读习惯
- **标签生成**: 自动生成相关话题标签，提升笔记曝光

## 输出格式

```
📌 标题选项:
1. 标题一
2. 标题二
3. 标题三

---

正文内容（含 emoji 排版）

---

🏷️ 标签:
#标签1 #标签2 #标签3
```
