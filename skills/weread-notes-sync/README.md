# 读书笔记同步

读书笔记同步是一个把「微信读书」个人阅读数据沉淀到静态网站的工具。它从微信读书 Agent API Gateway 拉取你的阅读统计、全部有笔记的书的划线与想法，自动生成一套静态 HTML 页面：一个书架封面网格列表页，加上每本书一个的笔记详情页。每条划线和想法都带深链，在手机上点击可直接跳回微信读书 App 的精确位置。

页面风格对齐 yuanchu.ai（纯白底、细体大字、极简排版），可直接并入个人网站作为「阅读」栏目。

## 使用场景

- 把多年微信读书的划线与想法沉淀成个人网站的长期内容资产
- 给个人博客/主页新增一个「阅读」栏目，展示书架与读书笔记
- 定期（手动或定时任务）刷新读书笔记，保持内容最新
- 把"读到某处的想法"以可检索、可分享的网页形式留存

## 特点

- **个人数据全量拉取**: 阅读统计（时长/天数/偏好领域）+ 全部有笔记的书的划线与想法
- **书架式列表页**: 封面网格 + 阅读统计概览，hover 上浮的书架质感
- **每本书独立详情页**: 想法（突出 + 附关联原文划线）与划线分区呈现
- **精确深链回跳**: 每条笔记用官方 `weread://bestbookmark` 格式跳回 App 对应划线位置，无位置信息时自动降级到章节/书级别
- **风格对齐**: 生成的页面与 yuanchu.ai 设计语言一致，开箱并入
- **安全**: API Key 仅通过环境变量 `WEREAD_API_KEY` 读取，不写入文件、不进仓库；网关只读，不改动微信读书账号

## 前置条件

- 微信读书 Agent API Gateway 密钥（格式 `wrk-xxxx`），通过 `export WEREAD_API_KEY=wrk-...` 提供
- Python 3（仅标准库）

## 输出格式

- `reading/index.html` — 书架列表页（全部书封面网格 + 阅读统计 + 偏好领域）
- `reading/book-<bookId>.html` — 每本书的笔记详情页（全部想法与划线，含深链）

## 使用方法

在 Claude Code 中说「更新读书笔记」或「同步微信读书」即可触发。
