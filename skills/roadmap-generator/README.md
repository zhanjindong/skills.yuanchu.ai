# 时间轴路标图生成

从 Excel/CSV 文件或用户描述中提取多维度路标数据，生成正式的时间轴路标图 HTML 页面。采用表格式泳道布局，按维度分行、按版本/时间节点分列，适合工作汇报、项目规划展示和投屏演示。

核心设计原则是"数据忠实、结构清晰"——原始描述内容保持不动，仅做维度归类和可视化呈现。

## 使用场景

- 产品版本路标规划展示
- 项目里程碑多维度对比
- 年度/季度工作规划汇报
- 历史事件多维度时间轴梳理
- 技术演进路线图

## 特点

- **数据忠实**: 原始描述文字保持不动，不做删减或改写，仅做维度归类
- **多维度泳道**: 支持 2-6 个维度并行展示，每个维度独立配色
- **时间轴布局**: 版本号 + 日期 + 状态徽章，节点连线清晰
- **正式风格**: 白底表格布局，适合正式工作汇报和打印
- **Excel 支持**: 可直接从 Excel/CSV 文件解析数据
- **垂直居中**: 维度标签使用 flex 居中方案，无论内容多少都保持对齐
- **打印友好**: 内置 print 媒体查询，适合导出 PDF

## 输出格式

生成单文件 HTML 路标图，包含：
- 标题区域（主标题 + 更新日期）
- 时间轴表头（版本号、日期、状态徽章）
- 多维度内容行（色块标签 + 有序列表内容）
- 空版本占位（破折号）
