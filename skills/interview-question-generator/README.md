# 面试提问

面试提问是一个面向复试面试官的智能面试问题生成工具。基于候选人简历内容，自动生成覆盖 5 大考察维度的结构化面试问题 HTML 页面，每个问题下方带有文本框方便面试官实时记录。

## 使用场景

- 复试前快速准备针对性面试问题，避免泛泛而谈
- 面试过程中在每个问题下方实时记录候选人回答要点
- 识别简历中的模糊描述和疑点，生成验证性追问
- 面试结束后打印/导出包含记录的完整面试评估

## 特点

- **HTML 输出**: 生成简洁专业的单文件 HTML 页面，浏览器直接打开即用
- **实时记录**: 每个问题下方带文本框，面试中随时记录候选人表现
- **5 维度覆盖**: 专业能力与经验、业务知识、学习能力与 AI 素养、求职动机与诉求、核心通用能力
- **简历驱动**: 每个问题都关联简历中的具体内容，拒绝通用模板
- **双路径追问**: 每个问题提供"回答顺利"和"回答受阻"两条追问方向
- **红旗检测**: 自动识别简历疑点，生成验证性追问清单
- **节奏建议**: 根据面试时长（默认 30 分钟）自动分配各维度时间
- **打印友好**: 支持打印/导出 PDF，保留所有记录内容

## 输出示例

生成的 HTML 页面包含：

- **顶部导航栏**: 候选人信息、操作按钮（清空记录、打印导出）
- **面试节奏表**: 各阶段时间分配
- **5 个维度区块**: 每个问题卡片含问题内容、关联简历、考察意图、期望要点、追问方向、记录文本框
- **红旗追问清单**: 简历疑点验证表
- **综合评价区**: 底部大文本框记录整体评价
