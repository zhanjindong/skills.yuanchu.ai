# 代码解读

代码解读是一个代码解释工具，它能用通俗易懂的语言解释复杂代码逻辑，帮助开发者快速理解陌生代码库或高级编程概念。

## 使用场景

- 接手他人项目时快速理解核心模块的实现逻辑
- 学习新框架或语言时，理解示例代码中的高级特性和设计模式
- 阅读开源项目源码时，解析难以理解的算法或架构设计
- 辅助代码审查，确保审查者正确理解被审查代码的意图

## 特点

- **分层解释**: 先给出一句话总结（这段代码做了什么），再逐步深入细节
- **类比教学**: 使用日常生活中的类比来解释抽象的编程概念，降低理解门槛
- **逻辑可视化**: 对于复杂流程，用伪代码或流程图辅助说明执行顺序
- **术语标注**: 遇到专业术语时自动给出简短解释，对初学者友好
- **多语言覆盖**: 支持解释 Python、JavaScript、TypeScript、Java、Go、Rust 等主流语言的代码

## 解释结构

1. 一句话总结 -- 这段代码的目的
2. 整体思路 -- 用类比或简单语言描述算法/设计思路
3. 逐段分析 -- 关键代码块的详细解释
4. 关键概念 -- 涉及的编程概念或设计模式说明
