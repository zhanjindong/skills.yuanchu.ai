# Explain Code — Claude Code Slash Command

## 使用方法

在 Claude Code 中输入 `/explain` 并指定文件路径或粘贴代码片段即可触发。

---

用通俗易懂的语言解释指定代码的逻辑，适合学习和快速理解陌生代码。

## 步骤

1. 读取用户指定的代码文件或代码片段。
2. 识别代码的编程语言和整体结构。
3. 按以下结构输出解释：

```markdown
## 一句话总结
[用一句话说明这段代码做了什么]

## 整体思路
[用日常类比或简单语言描述整体设计思路]

## 逐段分析

### [代码块1的功能描述]
- 代码位置: 第 X-Y 行
- 做了什么: [具体解释]
- 为什么这么做: [设计意图]

### [代码块2的功能描述]
...

## 关键概念
- **[术语1]**: [简短解释]
- **[术语2]**: [简短解释]
```

## 规则

- 解释面向"能编程但不熟悉这段代码"的读者，不要过于基础也不要跳过关键步骤
- 优先使用中文解释，代码术语保留英文（如 callback、Promise、middleware）
- 遇到设计模式（如观察者、策略、工厂）时明确指出并简要说明
- 对于复杂控制流（递归、回调嵌套、异步链），用"执行顺序"视角解释
- 如果代码有明显问题或不推荐的写法，在解释时顺带指出
- 不要逐行翻译代码，而是按逻辑块组织解释
