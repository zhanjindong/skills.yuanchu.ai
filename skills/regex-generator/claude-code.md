# 正则表达式生成 — Claude Code

## 使用方法

在 Claude Code 中输入 `/regex-generator` 即可触发。用自然语言描述你需要匹配的模式，自动生成正则表达式。

---

根据用户的自然语言描述，生成正则表达式并附带解释和测试用例。

## 步骤

1. 理解用户描述的匹配需求，明确：
   - **匹配目标**: 需要匹配什么样的字符串模式
   - **使用场景**: 验证、提取、替换
   - **编程语言**: JavaScript / Python / Java / Go / 通用（默认通用）
2. 生成正则表达式，优先选择可读性好且性能合理的写法。
3. 逐段解释正则表达式的每个部分。
4. 生成至少 3 个匹配和 3 个不匹配的测试用例。
5. 如果需求有歧义，提出澄清问题后再生成。

## 规则

- 优先使用简洁、可读的正则写法，避免过度复杂
- 明确说明正则的局限性（如无法处理嵌套结构）
- 测试用例要覆盖边界情况
- 如果不同编程语言的正则语法有差异，明确标注
- 对于常见模式（邮箱、手机号、URL），使用业界通用的成熟方案
- 如果用户需求过于复杂，建议拆分为多个正则或使用解析器
