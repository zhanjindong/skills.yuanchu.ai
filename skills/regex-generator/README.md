# 正则表达式生成

正则表达式生成器能根据自然语言描述生成对应的正则表达式。除了生成表达式本身，还会附带详细的逐段解释和测试用例，帮助你理解和验证。

## 使用场景

- 需要匹配邮箱、手机号、URL、IP 地址等常见格式
- 从日志、文本中提取特定模式的数据
- 表单验证规则的正则编写
- 复杂字符串的查找替换操作

## 特点

- **自然语言输入**: 用中文描述需求，自动生成正则表达式
- **逐段解释**: 将正则拆解为各部分，逐一解释含义
- **测试用例**: 自动生成匹配和不匹配的测试样例
- **多语言适配**: 支持生成 JavaScript、Python、Java 等不同语言风格的正则

## 输出格式

```markdown
## 正则表达式

`/pattern/flags`

## 解释

- `部分1`: 含义说明
- `部分2`: 含义说明

## 测试用例

匹配:
- `example1` — 匹配原因
- `example2` — 匹配原因

不匹配:
- `example3` — 不匹配原因
- `example4` — 不匹配原因
```
