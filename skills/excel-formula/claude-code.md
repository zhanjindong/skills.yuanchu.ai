# Excel 公式助手 — Claude Code

## 使用方法

在 Claude Code 中输入 `/excel-formula` 即可触发。用自然语言描述你的数据处理需求，自动生成公式。

---

根据用户的自然语言描述，生成 Excel 或 Google Sheets 公式并附带解释和示例。

## 步骤

1. 理解用户的数据处理需求：
   - **操作类型**: 查找匹配 / 条件统计 / 文本处理 / 日期计算 / 数据清洗 / 其他
   - **数据结构**: 数据所在的列/行、数据类型
   - **预期结果**: 希望得到什么样的输出
   - **平台**: Excel / Google Sheets / WPS（默认 Excel）
2. 生成最合适的公式，优先选择：
   - 新版函数（如 XLOOKUP 优于 VLOOKUP，需标注版本要求）
   - 可读性好的写法
   - 性能合理的方案
3. 逐步解释公式中每个函数和参数的作用。
4. 提供示例数据和预期结果表格。
5. 标注注意事项和平台兼容性。

## 规则

- 如果有多种实现方式，给出推荐方案并说明替代方案
- 明确标注函数的版本要求（如 XLOOKUP 需要 Excel 365/2021+）
- 处理可能的错误情况（如 VLOOKUP 找不到值时用 IFERROR 包裹）
- 涉及区域引用时，说明是否需要绝对引用（$）
- 标注 Excel 和 Google Sheets 的函数差异
- 复杂公式建议拆分为辅助列，降低维护难度
