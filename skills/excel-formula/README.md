# Excel 公式助手

Excel 公式助手能根据自然语言描述生成 Excel 或 Google Sheets 公式。支持从简单的求和统计到复杂的 VLOOKUP、INDEX-MATCH、数据透视等高级操作，并附带使用说明和示例。

## 使用场景

- 需要编写 VLOOKUP、INDEX-MATCH 等复杂查找公式
- 数据清洗中的文本处理、日期计算、条件统计
- 多条件汇总、动态范围引用等高级数据分析
- Google Sheets 特有函数（QUERY、IMPORTRANGE 等）的使用

## 特点

- **自然语言输入**: 用中文描述需求，自动生成对应公式
- **逐步解释**: 拆解公式中每个函数的作用和参数含义
- **示例演示**: 提供示例数据和预期结果，帮助理解公式效果
- **兼容提示**: 标注 Excel 和 Google Sheets 之间的函数差异

## 输出格式

```markdown
## 公式

`=公式内容`

## 解释

- `函数1(参数)`: 作用说明
- `函数2(参数)`: 作用说明

## 使用示例

| 列A | 列B | 结果 |
|-----|-----|------|
| 示例数据 | 示例数据 | 预期结果 |

## 注意事项
- 使用前提和限制条件
- Excel / Google Sheets 兼容性说明
```
