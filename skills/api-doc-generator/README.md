# API 文档生成

API 文档生成是一个自动化 API 文档生成工具。它能够分析后端接口代码（Controller、路由处理函数等），自动提取接口信息并生成结构清晰、内容完整的 API 文档。

## 使用场景

- 项目初期快速生成 API 文档，供前后端协作参考
- 已有项目补充缺失的接口文档，减少手动编写的工作量
- 接口变更后快速更新文档，保持文档与代码的一致性
- 生成标准化的 API 参考手册，供外部开发者集成使用

## 特点

- **自动提取**: 从代码注解、类型定义、参数校验规则中自动提取接口信息，无需额外配置
- **多框架支持**: 支持 Spring Boot、Express、FastAPI、Gin 等主流后端框架
- **完整覆盖**: 生成的文档包含请求方法、路径、参数说明、请求体示例、响应示例、错误码和认证要求
- **多格式输出**: 支持 Markdown、OpenAPI 3.0 YAML 两种输出格式
- **智能推断**: 即使代码注释不完整，也能根据命名和类型信息推断参数含义

## 输出内容

每个接口文档包含：HTTP 方法与路径、功能描述、请求参数（Query/Path/Body）、请求示例、成功响应示例、错误响应说明、认证要求。
