# 会议纪要生成

会议纪要生成是一个智能会议记录整理工具。它能从会议录音转写文本、聊天记录或手动笔记中，自动提取关键决策、行动项、负责人和截止日期，生成结构化的会议纪要。

## 使用场景

- 团队周会、项目评审会后快速生成可分发的会议纪要
- 从冗长的会议记录中提取核心决策和待办事项，确保信息不遗漏
- 跨部门会议后统一输出格式，方便归档和追踪执行情况

## 特点

- **智能提取**: 自动识别会议中的决策、行动项、风险和待确认事项
- **结构化输出**: 按议题归类，每个行动项标注负责人和截止日期
- **多格式输入**: 支持会议录音转写文本、IM 聊天记录、手写笔记等
- **可追踪**: 行动项输出为 checklist 格式，便于后续跟进

## 输出格式

```markdown
# 会议纪要

**会议主题**: xxx
**日期**: xxxx-xx-xx
**参会人**: A, B, C

## 关键决策
1. 决策内容

## 行动项
- [ ] 行动项描述 — @负责人 — 截止日期

## 讨论要点
- 议题及讨论内容摘要

## 待确认事项
- 需要进一步确认的问题
```
