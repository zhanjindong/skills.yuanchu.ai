# 会议纪要生成 — Claude Code

## 使用方法

在 Claude Code 中输入 `/meeting-summary` 即可触发。将会议记录文本粘贴或指定文件路径，自动生成结构化会议纪要。

---

根据用户提供的会议记录（录音转写、聊天记录、笔记等），提取关键信息并生成结构化会议纪要。

## 步骤

1. 接收用户提供的会议原始记录，判断输入类型（转写文本、聊天记录、笔记）。
2. 通读全文，识别并提取：
   - **会议基本信息**: 主题、日期、参会人
   - **关键决策**: 明确达成共识的结论
   - **行动项**: 需要执行的任务，标注负责人和截止日期
   - **讨论要点**: 各议题的核心讨论内容摘要
   - **待确认事项**: 未达成共识或需后续跟进的问题
3. 按照标准模板输出结构化会议纪要。
4. 如果原文中缺少负责人或截止日期，用 `[待指定]` 标注并提醒用户补充。

## 规则

- 行动项必须具体、可执行，避免模糊描述
- 每个行动项必须包含负责人和截止日期（或标注待指定）
- 忠实于原始记录，不编造未讨论的内容
- 讨论要点保留关键分歧和理由，不只记结论
- 输出使用中文，除非用户要求其他语言
