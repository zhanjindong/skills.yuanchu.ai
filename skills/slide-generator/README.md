# 幻灯片生成

幻灯片生成器能够根据主题或大纲，生成精美的单文件 HTML 幻灯片演示文稿。采用卡片式设计和流畅的导航交互，支持键盘、触摸和鼠标操作，生成的文件可直接在浏览器中全屏演示。

核心设计原则是"每张 slide 只做一件事"——一个要点对应一张 slide，宁可多加一张也不让单页内容过满。支持多种布局组件（卡片网格、对比布局、流程图、金字塔等），适配工程师、管理层、学生等不同受众。

## 使用场景

- 技术分享和内部培训课件制作
- 产品介绍和方案汇报演示文稿
- 学术报告和论文答辩 slides
- 会议演讲和外部分享材料
- 快速将文字大纲转化为可视化演示

## 特点

- **单文件 HTML**: 所有样式、脚本、内容打包在一个 HTML 文件中，无外部依赖，可离线使用
- **多种布局组件**: 卡片网格（g2/g3/g4）、高亮框、定义框、对比布局（VS）、流程图、金字塔、场景卡片等
- **流畅导航**: 支持键盘左右键、触摸滑动、鼠标点击，带侧边导航菜单和页码指示
- **渐入动画**: 支持 fade-in 逐步展示效果，增强演示体验
- **章节标签**: 按章节循环使用蓝/紫/绿/琥珀/红标签颜色，视觉层次清晰
- **响应式设计**: 适配不同屏幕尺寸
- **受众适配**: 工程师（技术细节多）、管理层（结论前置、数字驱动）、学生（概念清晰、类比丰富）

## 输出格式

生成单文件 HTML 幻灯片，包含：
- 封面 slide（主标题、副标题、章节快速跳转链接）
- 内容 slides（按章节组织，每章 3-8 张）
- 结尾 slide（总结 / Q&A）
- 侧边导航菜单（支持章节和子节跳转）
- 页码指示器

文件命名：`[主题]-slides_YYYY年MM月DD日.html`
