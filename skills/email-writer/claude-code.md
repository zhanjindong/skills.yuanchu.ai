# 邮件写作助手 — Claude Code

## 使用方法

在 Claude Code 中输入 `/email-writer` 即可触发。描述邮件场景、目的和关键信息，自动生成专业邮件。

---

根据用户提供的场景、目的和关键信息，生成结构清晰、措辞得体的专业邮件。

## 步骤

1. 确认用户的邮件需求：
   - **场景**: 商务合作 / 求职申请 / 客户沟通 / 内部协作 / 其他
   - **目的**: 邮件要达成的核心目标
   - **关键信息**: 需要包含的具体内容
   - **语气**: 正式 / 友好 / 简洁（默认根据场景自动判断）
   - **语言**: 中文 / 英文（默认中文）
2. 根据场景选择对应的邮件结构模板。
3. 组织邮件内容，包含：主题行、称呼、开头（背景/目的）、正文（要点）、结尾（行动呼吁）、署名。
4. 输出完整邮件，并标注可能需要用户自行替换的占位信息。

## 规则

- 主题行简洁明了，不超过 20 个字
- 正文段落层次清晰，每段聚焦一个要点
- 避免冗余客套话，直奔主题
- 商务英文遵循国际惯例（Dear / Best regards 等）
- 敏感信息用 `[xxx]` 占位，提醒用户替换
- 邮件长度控制在合理范围，避免过长
