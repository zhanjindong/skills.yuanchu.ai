# 邮件写作助手

邮件写作助手能根据场景和意图快速生成专业邮件。支持商务沟通、求职申请、客户跟进、内部协作等多种场景，自动调整语气和格式。

## 使用场景

- 商务合作邮件撰写，需要得体的措辞和清晰的诉求
- 求职场景的 Cover Letter 或自我推荐信
- 客户服务中的投诉回复、方案推荐、项目更新等
- 跨部门协作中的需求沟通、进度同步、会议邀请

## 特点

- **场景模板**: 内置商务、求职、客户沟通、内部协作等模板，一键切换
- **语气可调**: 支持正式、友好、简洁等多种语气风格
- **结构清晰**: 自动组织开头问候、正文要点、结尾呼吁行动的标准结构
- **多语言**: 支持中英文邮件，商务英文自动遵循国际惯例

## 输出格式

```
主题: xxx

收件人称呼,

开头段落（背景/目的）

正文段落（核心内容/要点）

结尾段落（行动呼吁/期待回复）

署名
```
