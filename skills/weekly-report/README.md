# 周报生成

周报生成工具能从工作记录、Git 提交日志、任务系统等多种数据源自动生成结构化周报。支持按项目、按人员、按优先级等多种组织方式，适配不同团队的周报风格。

## 使用场景

- 开发人员根据 Git 提交记录快速生成技术周报
- 项目经理汇总团队成员工作产出，生成项目周报
- 个人整理一周工作内容，高效完成周报撰写

## 特点

- **多数据源**: 支持从 Git log、任务列表、会议记录、手动输入等多种来源提取信息
- **智能归类**: 自动将零散的工作项按项目或模块归类整理
- **风格可调**: 支持简洁版（要点式）、详细版（叙述式）、汇报版（面向管理层）
- **量化呈现**: 自动统计代码提交数、完成任务数等量化指标

## 输出格式

```markdown
# 周报 — YYYY.MM.DD ~ YYYY.MM.DD

## 本周完成
- [项目A] 完成内容描述
- [项目B] 完成内容描述

## 进行中
- [项目A] 进行中的工作及进度

## 下周计划
- 计划事项及预期产出

## 风险与问题
- 需要协助或关注的事项
```
