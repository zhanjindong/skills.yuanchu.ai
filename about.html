About Skills Market

AI Skills 集市，让你的 Agent 更强大

PLATFORM

Skills Market 是元初AI旗下的 AI 技能集市，致力于收集和展示各领域优质的 AI Skills。

AI Agent 生态快速发展，Claude Code、Codex、OpenClaw 等 Agent 框架各有其技能与插件格式。开发者创造了大量有趣实用的 Skills，但缺少一个统一的展示和发现平台。

Skills Market 汇聚这些优质技能，同一 Skill 支持多种 Agent 框架格式下载，让用户一站找到所需技能，让你的 Agent 更强大。

SUPPORTED FRAMEWORKS

Claude Code
Anthropic 的 CLI 编程助手，通过 slash command 和 CLAUDE.md 扩展能力。
format: .md (slash command / CLAUDE.md skill)
Codex
OpenAI 的编程 Agent，支持通过配置文件扩展技能。
format: .md / .yaml
OpenClaw
开源 Agent 框架，支持灵活的插件系统。
format: .json / .yaml

更多框架持续接入中，欢迎提交适配请求。

SUBMIT A SKILL

我们欢迎社区贡献优质的 AI Skills。提交流程如下:

在 skills/ 目录下创建以 Skill ID 命名的目录（使用 kebab-case 命名）
编写 README.md（详细功能描述）和至少一个框架格式的 Skill 实现文件
在 data/skills.json 中添加一条元数据记录，formats[].file 指向对应文件路径
通过 GitHub Pull Request 提交，管理员审核后即可上线
